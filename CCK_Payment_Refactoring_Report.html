✅ 수동 테스트 체크리스트

1. 로고/프로필 정렬 확인

  • 헤더에서 로고가 좌측에 정렬되어 있는지 확인
  • 프로필 아이콘과 '로그인' 텍스트가 우측에 정렬되어 있는지 확인
  • 로고와 텍스트 사이 간격이 적절한지 확인

2. 반응형 브레이크포인트 확인

  • 1280px: 데스크톱 정상 표시
  • 1024px: 테이블 가로 스크롤 동작 확인
  • 768px: 헤더 세로 정렬, 테이블 카드형 변환 확인
  • 375px: 모바일 최적화 레이아웃 확인

3. Tab키 포커스 순서/스타일 확인

  • Tab 키로 로그인 버튼 포커스 이동 확인
  • 포커스 시 파란색 아웃라인 표시 확인
  • Enter/Space 키로 로그인 버튼 활성화 확인
  • 테이블 헤더 셀 포커스 스타일 확인

4. 이미지 CLS 유무 확인

  • 페이지 로딩 시 레이아웃 시프트 없음 확인
  • 로고 이미지 즉시 표시되는지 확인
  • 프로필 아이콘 크기 고정되어 있는지 확인

5. Favicon/타이틀/메타 확인

  • 브라우저 탭에 CCK 로고 favicon 표시 확인
  • 타이틀: "큐빙클럽코리아 - 루빅스 큐브 대회 접수 및 결제" 확인
  • meta description 검색엔진 노출 시 표시 확인
  • 개발자 도구에서 meta keywords 존재 확인

6. 로딩/에러 상태 확인

  • ?dummy=false URL로 접속 시 "행사가 없습니다" 메시지 확인
  • 네트워크 에러 시뮬레이션으로 에러 메시지 확인
  • 로딩 중 "데이터를 불러오는 중입니다..." 메시지 확인
